📚 Study Tracker – Virtual City E-Commerce Project

A structured study roadmap to build a 3D Virtual City E-Commerce Platform using modern full-stack technologies and 3D web development.

This repository tracks all the topics, progress, and learning phases required before developing the final large-scale project.


🎯 Project Vision

To build an interactive virtual city 🏙️ where:

Users can explore a 3D environment
Buildings represent different shops
Users can interact (e.g., press "E") to enter shops
Purchase products through an integrated e-commerce system

🧠 Objective

Before building the project, this tracker helps to:

📌 Cover all required technologies step-by-step
📌 Maintain structured learning
📌 Track progress level-by-level
📌 Avoid gaps in knowledge
